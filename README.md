# App-SESMT-EPI-Demo
App Android para entrega e controle de EPI/EPC (conformidade NR-6). Demonstração desenvolvida para o time de Segurança do Trabalho: digitaliza o processo de entrega, coleta assinatura eletrônica, valida CA, histórico e relatórios. Modo offline + integração SQL Server. Versão demo para portfólio.
